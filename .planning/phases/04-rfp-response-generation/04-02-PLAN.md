    The POST handler must:

    1. **Auth guard:** Check `await auth()` for session.user.id. Return 401 if missing.

    2. **Parse request body:** Extract `{ caseId, documentId, startFrom }` from JSON body. `startFrom` is optional (for D-13 continue generation).

    3. **Fetch extracted requests:** Query `extractedRequests` where documentId matches. If `startFrom` is provided, filter to only requests with `requestNumber >= startFrom`. Sort by requestNumber ascending.

    4. **Fetch PDF documents:** Query `documents` where caseId matches to find the RFP document (by documentId) and any complaint document (type === "complaint"). Fetch both PDFs from Vercel Blob using Bearer token auth (`process.env.BLOB_READ_WRITE_TOKEN`), base64-encode them.

    5. **Build Claude message:** Construct the messages array:
       - Content blocks: complaint PDF document block (if exists) + RFP PDF document block + text block with `buildGenerationPrompt(requests, hasComplaint)`
       - Use `client.messages.stream()` with:
         - model: "claude-sonnet-4-5" (per D-12)
         - max_tokens: 16384 (per Pitfall 1)
         - output_config using the Zod schema via manual JSON schema or zodOutputFormat

    6. **Stream SSE to client (hybrid approach per RESEARCH.md Pattern 2):**
       - Create a `ReadableStream` with a `start(controller)` callback
       - On `stream.on("text", (textDelta, textSnapshot))`: count `"request_number"` occurrences in textSnapshot for progress. Enqueue SSE event: `data: {"type":"progress","count":N,"total":T}\n\n`
       - After `stream.finalMessage()`: parse the full response text with `generationResponseSchema.parse()`, then bulk-persist all responses to the `generatedResponses` table. Enqueue SSE event: `data: {"type":"complete","responses":[...]}\n\n`
       - On error: attempt to parse whatever JSON was accumulated for partial save (D-13). Persist any parseable responses. Enqueue SSE event: `data: {"type":"error","message":"...","savedCount":N}\n\n`
       - On stream end: enqueue `data: [DONE]\n\n` and close controller.

    7. **Return Response** with headers:
       ```
       Content-Type: text/event-stream; charset=utf-8
       Cache-Control: no-cache, no-transform
       Connection: keep-alive
       ```

    **DB persistence logic:**
    For each response in the parsed array, insert into generatedResponses:
    - Find the matching extractedRequest by requestNumber
    - Set requestId to the matching extractedRequest.id
    - Set pattern, responseText, crossReferenceNumber from the parsed response
    - Set objectionTypes to JSON.stringify(response.objection_types) if not null, else null

    **Partial save on error (D-13):**
    Wrap the finalMessage/parsing in try-catch. On error:
    - Get the accumulated text snapshot from the stream
    - Attempt to extract any complete response objects from the partial JSON
    - Persist whatever was parsed
    - Send error event with savedCount

    **Step 1: Create deleteGeneratedResponses server action**

    Create `src/actions/generate.ts`:

    ```typescript
    "use server";

    import { db } from "@/lib/db";
    import { generatedResponses, extractedRequests } from "@/lib/db/schema";
    import { eq, inArray } from "drizzle-orm";
    import { auth } from "@/lib/auth";
    import { revalidatePath } from "next/cache";

    export async function deleteGeneratedResponses(
      documentId: string,
      caseId: string
    ): Promise<{ success: boolean; error?: string }> {
      const session = await auth();
      if (!session?.user?.id) throw new Error("Not authenticated");

      try {
        // Find all extracted requests for this document
        const requests = await db.query.extractedRequests.findMany({
          where: eq(extractedRequests.documentId, documentId),
        });

        const requestIds = requests.map((r) => r.id);

        if (requestIds.length > 0) {
          await db
            .delete(generatedResponses)
            .where(inArray(generatedResponses.requestId, requestIds));
        }

        revalidatePath(`/case/${caseId}`);
        return { success: true };
      } catch (error) {
        console.error("Failed to delete generated responses:", error);
        return { success: false, error: "Failed to delete responses" };
      }
    }
    ```
